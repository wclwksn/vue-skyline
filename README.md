# vue-skyline

> 使用vue集成skyline的三维球控件

#### 实现
+ TE的控件放在App.vue中，也可以制作vue的组件进行集成。
+ 测试了一个弹出窗口的效果。
+ 
## Build Setup
### vue基本运行过程

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
